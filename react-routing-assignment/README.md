#  React Combined Assignment (Vite)

This project combines **React Basics** and **React Routing** concepts in a single application built using **Vite + React**.

---

##  Features

###  React Basics
- Reusable components
- Props implementation (Product component)
- State management using `useState` (Counter)
- Dynamic UI updates

###  React Routing
- Multi-page navigation using React Router
- Pages: Home, About, Contact
- 404 Page handling
- Smooth navigation without page reload

---

##  Tech Stack

- React (Vite)
- React Router DOM
- CSS (Custom styling)
