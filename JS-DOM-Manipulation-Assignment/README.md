#  To-Do List App

A simple **To-Do List application** built using **HTML, CSS, and JavaScript** to practice **DOM Manipulation**.

## Features
- Add, edit, and delete tasks  
- Save tasks using localStorage  
- Clean and responsive UI  

##  Tech Stack
- HTML  
- CSS  
- JavaScript  

##  Learning
- JavaScript DOM manipulation  
- Dynamic UI updates  
- localStorage usage
